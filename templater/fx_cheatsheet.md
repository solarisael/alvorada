# Text FX Cheatsheet

Use canonical markers in notes:

`{{fx:effect_name}}your_text{{/fx}}`

## Effects

- `glow`
- `neon`
- `shadow`
- `chroma`
- `blur`
- `flicker`
- `rainbow`
- `gradient`
- `wiggle`
- `float`
- `shake`
- `glitch`

## Quick Examples

```md
{{fx:glow}}luminous line{{/fx}}
{{fx:neon}}ritual signal{{/fx}}
{{fx:chroma}}split echo{{/fx}}
{{fx:flicker}}fading lantern{{/fx}}
{{fx:glitch}}fractured memory{{/fx}}
```

## Fast Workflow

1. Select text.
2. Trigger `FX Picker` (QuickAdd).
3. Choose effect.
4. Keep markers lightweight (phrase-level emphasis).
